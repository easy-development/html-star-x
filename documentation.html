Information About The Files

Every Section of the page is found in the parts folder. And Each part has it's own LESS CSS File.

If you want to edit the template easily, you should use the LESS files & compile them afterwards.

Else, you can use the assets/style/style.css and edit everything from there also.

Regarding the images, all the images are found in assets/image
